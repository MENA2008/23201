Our Project

the problem the project attempts to solve is using polluted air from industrial waste and converting it into clean air. The prototype’s target is to get clean air for breathing from the input air polluted by “Methane”.  This prototype was made with three phases, the first one is to medical gauze that captures large particles, then passes through activated carbon, and then to fine filter to ensure that all impurities are gone completely. This project has the potential to successfully meet the design requirements. When the prototype was tested, it had a purification percentage of about 25% of the input air. Finally, this project would have an efficiency of more than 25%.
